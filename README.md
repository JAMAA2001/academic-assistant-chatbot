# Academic Chatbot (RAG)

Ce projet est un chatbot basé sur le principe du RAG (Retrieval-Augmented Generation),
développé avec Python et Streamlit.

⚠️ Certaines données utilisées par l'application sont **privées** et ne sont donc
pas incluses dans ce dépôt GitHub.

---

## Prérequis
- Python 3.9+
- pip
- Un compte avec une clé API Groq
